# Feature Specification - AI 塔罗文案生成器

## 概述

构建一个 AI 驱动的小红书文案生成工具，专门用于生成 AI 塔罗师（BT-7274）人设的内容，包含完整文案和配图提示词。

## 用户故事

### US1: 生成 AI 看塔罗文案
**作为** 内容创作者  
**我想要** 输入牌名自动生成完整文案  
**以便于** 快速产出高质量的塔罗解读内容

**验收标准**:
- [ ] 输入牌名（中文/英文）
- [ ] 输出完整文案（标题 + 正文 + 标签）
- [ ] 输出封面图提示词
- [ ] 输出 3-4 个内页图提示词
- [ ] 文案符合品牌人设（AI 视角）

### US2: 生成选牌测试文案
**作为** 内容创作者  
**我想要** 生成互动式选牌测试  
**以便于** 提高粉丝互动率

**验收标准**:
- [ ] 输出测试问题
- [ ] 输出 3-4 个选项（含牌名）
- [ ] 输出每个选项的详细解读
- [ ] 输出互动引导语
- [ ] 输出配套配图提示词

### US3: 生成运势文案
**作为** 内容创作者  
**我想要** 生成日/周运势  
**以便于** 维持固定栏目更新

**验收标准**:
- [ ] 支持日运/周运选择
- [ ] 输出 12 星座运势
- [ ] 输出整体能量描述
- [ ] 输出幸运指引
- [ ] 格式统一易读

### US4: 批量生成内容
**作为** 内容创作者  
**我想要** 批量生成多篇内容  
**以便于** 提前储备内容库存

**验收标准**:
- [ ] 支持批量输入主题
- [ ] 输出多篇完整文案
- [ ] 每篇独立文件存储
- [ ] 生成内容清单

---

## 功能需求

### FR1: 文案生成引擎
```
输入：
- 栏目类型（AI 看塔罗/塔罗看 AI/运势/哲思）
- 主题/牌名
- 可选参数（语气、长度、重点）

输出：
- 标题（含 emoji）
- 正文（结构化）
- 标签（8-12 个）
- 互动引导语
```

### FR2: 配图提示词生成
```
输入：
- 文案主题
- 图片类型（封面/内页）
- 风格偏好

输出：
- 封面图提示词 ×1
- 内页图提示词 ×2-4
- 每张图用途说明
```

### FR3: 模板管理
```
- 支持 4 个栏目的文案模板
- 支持配图提示词模板
- 模板可配置和扩展
- 支持自定义模板
```

### FR4: 内容导出
```
- 导出 Markdown 格式
- 导出 JSON 格式（结构化）
- 按日期/栏目组织目录
- 生成内容清单
```

## 非功能需求

### 性能
- 单篇文案生成 <5 秒
- 批量生成（10 篇）<30 秒
- 支持并发请求

### 质量
- 文案通顺无错别字
- 结构符合模板规范
- 提示词可被 AI 绘画工具理解

### 可维护性
- 模块化代码
- 完整文档
- 单元测试覆盖>80%
